🎵 App de Aulas de Instrumentos Musicais

Disciplina: Projeto Extensionista — Faculdade Uninter

Desenvolvido em parceria com: Rebeca (colega de curso)

Status: ✅ Concluído

Sobre o projeto

Desenvolvemos um protótipo de aplicativo voltado para o ensino de instrumentos musicais de forma online, conectando professores e alunos em um ambiente digital completo e interativo.

Funcionalidades do app

  • 🎬 Videoaulas gravadas para os alunos assistirem no próprio ritmo
  • 🔐 Login separado para professores e alunos
  • 📋 Mural de avisos para comunicados e atualizações da turma
  • 💬 Sala de bate-papo para alunos tirarem dúvidas em tempo real
  • 🎼 Acesso a partituras digitais dentro da plataforma

Objetivo

Democratizar o acesso ao ensino musical, oferecendo uma plataforma acessível, organizada e interativa para alunos e professores de instrumentos musicais.

🪙 Sistema Cofrinho de Moedas — Java

Disciplina: Programação Orientada a Objetos

Linguagem: Java

Status: ✅ Concluído

Sobre o projeto

Sistema que emula um cofrinho de moedas internacionais, desenvolvido com foco nos conceitos de herança e polimorfismo em Java.

Funcionalidades

  • ➕ Adicionar moedas de diferentes valores e países (Real, Dólar, Euro, etc.)
  • ➖ Remover moedas específicas do cofrinho
  • 📋 Listar todas as moedas armazenadas
  • 💱 Calcular o total convertido para Real Brasileiro

Conceitos aplicados

  • Herança: classe abstrata Moeda com subclasses Real, Dolar, Euro
  • Polimorfismo: cada moeda calcula sua conversão de forma independente
  • Coleções: uso de ArrayList para gerenciar as moedas

🐍 Calculadora de Plano de Saúde — Python

Disciplina: Lógica de Programação e Algoritmos

Linguagem: Python

Status: ✅ Concluído

Sobre o projeto

Programa que calcula o valor mensal de um plano de saúde com base na idade do cliente e no valor base informado, aplicando regras de desconto e acréscimo conforme faixas etárias.

Funcionalidades

  • 👤 Exibe mensagem de boas-vindas com nome e sobrenome
  • ⌨️ Leitura do valor base do plano e da idade via input
  • 📐 Aplicação de regras por faixa etária usando if, elif e else
  • 💰 Cálculo e exibição do valor mensal final
  • 📝 Código comentado seguindo as exigências da disciplina
